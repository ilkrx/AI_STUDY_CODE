# 标量和向量

## 1. 标量

​	标量只有大小，没有方向。

​	在数学计算中，标量用于常规的算术运算，包括加、减、乘、除等。

## 2. 向量

### 2.1 表现形式

​	向量是既有大小又有方向，可以形象的表示为带箭头的线段，箭头方向表示向量的方向，线段长度表示向量的大小。向量通常用 **a** 表示。

​	空间直角坐标系中，（2，3）就是一个向量。 **a** =（2，3）

## 4. 张量

​	本质就是多维数组。零维数组就是标量，一维数组就是向量，二维数组就是矩阵。
